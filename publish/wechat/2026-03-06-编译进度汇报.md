## 🔧 遇到的问题

### 1. HistoryManager 作用域

**问题**: SwiftUI Preview 中 EnvironmentObject 未定义

**解决**: 在 Preview 中添加完整的 environmentObject

### 2. enum 重复值

**问题**: ContainerType enum 有重复的 raw value

**解决**: 修改为不同的值

### 3. 文件删除后引用

**问题**: 删除 ContentView.swift 后项目仍引用

**解决**: 重新创建文件或清理项目

## 💡 经验总结

### 编译问题排查

1. **看错误信息** - 仔细阅读错误提示
2. **检查文件** - 确认文件存在且正确
3. **清理项目** - Clean Build Folder
4. **重新编译** - 很多时候能解决

### SwiftUI 常见坑

1. **EnvironmentObject** - 必须在顶层设置
2. **Preview** - 需要完整的 environment
3. **enum** - raw value 不能重复
4. **UserDefaults** - 没有 removeAll() 方法
